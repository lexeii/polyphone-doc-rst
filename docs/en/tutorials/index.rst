Tutorials

  |br| Learn the basics to create a new soundfont.

:ref:`custom releases`
  |br| The purpose of this tutorial is to create expressive instruments using looped samples whose releases have been recorded.
  You will learn how to fully use the capabilities of such a sample, by using the attack and sustain of the sample but also its release.

:ref:`how to prepare a sample`
  |br| An instrument built in a soundfont is made of samples, coming for example from the recordings of a real instrument.
  This tutorial describes how to add and prepare a sample in a soundfont.
