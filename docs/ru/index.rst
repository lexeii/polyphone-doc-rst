Добро пожаловать в документацию Polyphone!
==========================================

Polyphone — это бесплатное программное обеспечение с открытым исходным кодом для редактирования SoundFont форматов sf2, sf3, sfArk и sfz.
Простой интерфейс и удобные средства помогают эффективно работать как с маленькими так и с большими инструментами.
Polyphone доступен на разных языках для операционных систем Windows, Mac OS X, Linux. Имеется форум и подробная документация.
